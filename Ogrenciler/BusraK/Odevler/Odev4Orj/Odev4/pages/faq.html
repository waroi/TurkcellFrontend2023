2. Click the Generate Return Shipping Code button.
3. Select the item to be returned and the reason for the return. If you have purchased more than one product of the same product, you must also select the number of products to be returned.
4. Choose one of the suggested return methods according to the suitability of the products you want to return.
5. In case of more than one shipping option, select your cargo.
6. Take note of the return shipping code that appears on the screen. You can also find your return shipping code on the My Orders page.
7. Put the products with the same return code in the same package with the invoice and make sure that the package is intact.
8. Products with different return codes should be given in different packages, even if they are given to the same cargo.
9. According to the return method you choose, deliver your package to the cargo branch within 7 days or deliver it to the officer who will come to your address on the day of your appointment.
10. If you pass 7 days, you will need to get the return shipping code again.

If you have ordered the wrong product or changed your mind, you can easily cancel your products before the cargo preparations begin, as follows.
1. By following the steps “My Orders” > “Details/Order Detail” > “Cancel Order”.
2. By clicking "Connect to Live Help" from the "Help" step in the "My Account" section, selecting "Cancellation and Refund" on the page that opens. If your products have started to be prepared for shipping (your invoice has been issued) or if they have been shipped, they cannot be cancelled. In this case, you can return the product when you receive it.
CLUE: Fee refunds are completed within 2 business days. You will be notified by e-mail when your return is completed and your return is made to your bank.
